# Assets Plan

## Зачем нужен этот файл

Документ нужен:

- чтобы понимать, что именно нужно для сборки сцен;
- чтобы не делать арты хаотично;
- чтобы видеть приоритеты для первого играбельного куска;
- чтобы связать сцены, персонажей, фоны, CG и звук.

## First playable slice

Рекомендуемый стартовый demo-блок:

- `prologue_party.md`
- `scene_succubus_first_contact.md`
- `route_lust_intro.md`
- `route_romance_intro.md`
- `route_purity_intro.md`
- `first_morning_after.md`

Этот блок уже показывает:

- атмосферу;
- первый контакт;
- разницу маршрутов;
- бытовое продолжение.

## Ассеты по сценам

### prologue_party.md

- Фон входа / подъезда или общий внешний подход, если нужен.
- Фон квартиры / комнаты вечеринки.
- Фон кухни.
- Фон коридора.
- Персонажные спрайты:
    - ГГ;
    - Маша;
    - Артём;
    - Лера.
- Музыка: party / uneasy party.
- Возможный CG: момент ритуального предложения.

### scene_succubus_first_contact.md

- Фон ванной.
- Зеркало.
- Свеча.
- Особая тревожная вариация света.
- Звук / музыкальное состояние ритуала.
- Возможный CG:
    - отражение не совпадает;
    - первое ощущение присутствия.
- Ранний эффект присутствия суккубы.

### route_lust_intro.md

- Пост-ритуальный коридор / кухня / вечеринка.
- Варианты выражения лица героя.
- Лёгкий route-специфический эффект.
- Музыкальный оттенок соблазна.
- Акцентный CG не обязателен, но желателен позже.

### route_romance_intro.md

- Те же базовые фоны или вариации.
- Другой эмоциональный акцент.
- Мягкая тревога и внимательность.
- Музыка / саунд состояния наблюдения.

### route_purity_intro.md

- Те же фоны, но холоднее по ощущению.
- Акцент на сжатии, боли, сопротивлении.
- Звук / музыкальный оттенок отката и напряжения.

### first_morning_after.md

- Комната героя утром.
- Телефон.
- Зеркало / шкаф.
- Свет из окна.
- Возможный маленький CG: отражение / след на ладони.

### first_whispers.md

- Фон маршрутки.
- Стекло / отражение.
- Дневной городской фон.
- Небольшой звуковой акцент шёпота.

### school_tension.md

- Школьный коридор.
- Класс / дверь класса.
- Толпа / школьный шум.
- Спрайты Маши и Артёма в школьной среде.

### summer_free_days.md

- Сквер / фонтан.
- Улица.
- Набережная.
- Дневной и вечерний летний свет.
- Потенциальные дополнительные NPC silhouettes.

### investigation_intro.md

- Лавка / сквер.
- Телефон / фото / блокнот Маши.
- Спокойный детективный тон без нуара.

### artem_private_talk.md

- Двор / спортплощадка.
- Вечерний свет.
- Спрайт Артёма в более серьёзном состоянии.

### lera_followup.md

- Остановка вечером.
- Рекламный щит / стекло.
- Лера в повседневной одежде.
- Акцентное ощущение отсутствия отражения.
- Звуковой акцент напряжённого присутствия.

## Что делать первым

1. Главные персонажи первого акта.
2. Ключевые фоны first playable slice.
3. Музыкальные состояния first playable slice.
4. 1–2 самых важных CG.
5. Затем расширение на расследование и летние сцены.
