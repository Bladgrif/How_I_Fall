# Assets Plan

## Зачем нужен этот файл

Документ нужен:

- чтобы понимать, что именно нужно для сборки сцен;
- чтобы не делать арты хаотично;
- чтобы видеть приоритеты для первого играбельного куска;
- чтобы связать сцены, персонажей, фоны, CG и звук.

## First playable slice

Рекомендуемый стартовый demo-блок:

- `prologue_party.md`
- `scene_succubus_first_contact.md`
- `route_lust_intro.md`
- `route_romance_intro.md`
- `route_purity_intro.md`
- `first_morning_after.md`

Этот блок должен сразу задать:

- японскую школьную / городскую среду;
- лето;
- cinematic anime atmosphere;
- первый контакт;
- разницу маршрутов;
- мистику в обычном быту;
- бытовое продолжение после ритуала.

## Ассеты по сценам

### prologue_party.md

- Фон входа / подъезда или внешний подход к японской квартире / дому.
- Фон квартиры / комнаты вечеринки с подростковым хаосом.
- Фон кухни с напитками, упаковками из конбини, тесным светом.
- Фон коридора.
- Персонажные спрайты:
  - ГГ;
  - Маша;
  - Артём;
  - Лера.
- Музыка: party / uneasy party.
- Возможный CG: момент ритуального предложения.
- Важно: сразу считать японский летний городской быт.

### scene_succubus_first_contact.md

- Фон маленькой японской ванной / умывальной зоны.
- Зеркало.
- Свеча.
- Особая тревожная вариация света.
- Звук / музыкальное состояние ритуала.
- Возможный CG:
  - отражение не совпадает;
  - первое ощущение присутствия.
- Ранний эффект присутствия суккубы через стекло, отражение, свет и пустоту в кадре.

### route_lust_intro.md

- Пост-ритуальный коридор / кухня / вечеринка в японской квартире.
- Варианты выражения лица героя.
- Лёгкий route-специфический эффект: теплее свет, чуть острее взгляды.
- Музыкальный оттенок соблазна.
- Акцентный CG не обязателен, но желателен позже.

### route_romance_intro.md

- Те же базовые фоны или вариации.
- Другой эмоциональный акцент: мягкая тревога, внимательность, паузы.
- Важно: японская бытовая среда должна оставаться реальной, без резкого мистического перегруза.
- Музыка / саунд состояния наблюдения.

### route_purity_intro.md

- Те же фоны, но холоднее по ощущению.
- Акцент на сжатии, боли, сопротивлении.
- Звук / музыкальный оттенок отката и напряжения.
- Свет может стать более жёстким и пустым.

### first_morning_after.md

- Комната героя утром в японской квартире / доме.
- Телефон.
- Зеркало / шкаф.
- Мягкий свет из окна.
- Возможный маленький CG: отражение / след на ладони.
- Важно: обычное утро должно быть красивым, но уже треснувшим.

### first_whispers.md

- Фон маршрутки / автобуса / поезда.
- Стекло / отражение.
- Дневной японский городской фон.
- Небольшой звуковой акцент шёпота.
- Важно: мистика встроена в транспортную повседневность.

### school_tension.md

- Японский школьный коридор.
- Класс / дверь класса / обувные шкафчики.
- Толпа / школьный шум.
- Спрайты Маши и Артёма в школьной среде.
- Важно: social pressure через пространство, форму, взгляды и паузы.

### summer_free_days.md

- Сквер / маленький городской парк.
- Улица жилого района.
- Набережная / река / канал.
- Дневной и вечерний летний свет.
- Потенциальные дополнительные NPC silhouettes.
- Важно: ощущение длинного японского лета и пространства выбора.

### investigation_intro.md

- Лавка / сквер / маленький парк.
- Телефон / фото / блокнот Маши.
- Спокойный детективный тон без нуара.
- Важно: разговор должен оставаться подростковым и бытовым.

### artem_private_talk.md

- Двор / спортплощадка жилого района.
- Вечерний свет.
- Спрайт Артёма в более серьёзном состоянии.
- Важно: обычность места контрастирует со странностью разговора.

### lera_followup.md

- Остановка вечером / станция / городской переход.
- Рекламный щит / стекло.
- Лера в повседневной одежде.
- Акцентное ощущение отсутствия отражения.
- Звуковой акцент напряжённого присутствия.
- Важно: влажный вечерний воздух, городские огни, тихая неуместность Леры.

## Что нужно утвердить до массовой генерации арта

- Тип города: большой город, пригород, малый город у реки / станции.
- Школьная визуальная среда: форма, коридоры, класс, обувные шкафчики.
- Финальные имена или рабочая система имён.
- Степень реалистичности аниме-стиля.
- Визуальная дистанция между Лерой и суккубой.

## Что делать первым

1. Главные персонажи первого акта.
2. Ключевые фоны first playable slice.
3. Музыкальные состояния first playable slice.
4. 1–2 самых важных CG.
5. Затем расширение на расследование и летние сцены.
