# Audio Plan

## Зачем нужен этот файл

Документ нужен:

- чтобы музыка и звук работали по состояниям, а не по случайным трекам;
- чтобы не генерить музыку без задачи;
- чтобы сцены имели устойчивую атмосферу;
- чтобы заранее понимать, где нужна музыка, а где лучше тишина.

## Основные музыкальные состояния первого акта

### Main menu theme

- Ощущение: мистика + лето + подростковая драма.
- Тон: притягательный, немного тревожный.
- Не нужно: слишком громкий хоррор или эпический драматизм.

### Summer day theme

- Обычный город.
- Жара.
- Подростковая свобода.
- Лёгкая тревога под поверхностью.

### Party theme

- Живая подростковая тусовка.
- Шум и фоновая энергия.
- Музыка должна быть бытовой, но с возможностью лёгкой неправильности.

### Ritual tension theme

- Сжатие.
- Тишина.
- Внимание.
- Неправильность.
- Ощущение, что реальность смещается.

### Succubus whisper theme

- Интимно.
- Тихо.
- Тревожно.
- Притягательно.
- Не превращать в откровенный “сексуальный клише-трек”.

### School awkwardness theme

- Неловкость.
- Социальное давление.
- Подростковая уязвимость.
- Обычный день с внутренней трещиной.

### Investigation theme

- Спокойное напряжение.
- Сбор фактов.
- Тихий детектив.
- Без тяжёлого триллера.

### Warm human / Masha theme

- Опора.
- Честность.
- Тепло.
- Бытовая человеческая реальность.

### Wrongness stinger / short cues

Короткие сигналы для:

- отражения;
- сбоя звука;
- тихого шёпота;
- момента внутреннего провала;
- тонких звуковых сигналов странности.

## Какие музыкальные задачи нужны первыми

- Main menu.
- Party.
- Ritual tension.
- Whisper motif.
- Morning / summer day.
- School tension.
- Investigation light.

## Общие правила по звуку

- Не перегружать сцены музыкой.
- Мистика работает лучше через контраст с обычностью.
- Шёпоты и странности должны быть дозированными.
- У Маши и “человеческой линии” желательно свой более тёплый тон.
- Lust / romance / purity позже могут получить свои вариации одной общей мотивной базы.
