# Audio Plan

## Зачем нужен этот файл

Документ нужен:

- чтобы музыка и звук работали по состояниям, а не по случайным трекам;
- чтобы не генерить музыку без задачи;
- чтобы сцены имели устойчивую атмосферу;
- чтобы заранее понимать, где нужна музыка, а где лучше тишина.

На текущем этапе аудио-вектор: японское лето, подростковый городской быт, повседневность + мистика, тихая тревога и
cinematic anime tone.

## Основные музыкальные состояния первого акта

### Main menu theme

- Ощущение: мистика + лето в Японии + подростковая драма.
- Тон: притягательный, немного тревожный, cinematic anime.
- Не нужно: слишком громкий хоррор или эпический драматизм.

### Summer day theme

- Японский город / пригород.
- Жара.
- Цикады, улицы, станция, конбини, дневной воздух.
- Подростковая свобода.
- Лёгкая тревога под поверхностью.

### Party theme

- Живая подростковая тусовка в японской квартире / доме.
- Шум и фоновая энергия.
- Музыка должна быть бытовой, но с возможностью лёгкой неправильности.
- Не делать трек слишком клубным или взрослым.

### Ritual tension theme

- Сжатие.
- Тишина.
- Внимание.
- Неправильность.
- Ощущение, что реальность смещается.
- Хорошо работает через контраст с приглушённым шумом вечеринки за стеной.

### Succubus whisper theme

- Интимно.
- Тихо.
- Тревожно.
- Притягательно.
- Не превращать в откровенный “сексуальный клише-трек”.
- Важно: ощущение близости внутри головы, а не внешней атаки.

### School awkwardness theme

- Японская школьная среда.
- Неловкость.
- Социальное давление.
- Подростковая уязвимость.
- Обычный день с внутренней трещиной.

### Investigation theme

- Спокойное напряжение.
- Сбор фактов.
- Тихий детектив.
- Летний город вокруг остаётся живым.
- Без тяжёлого триллера и нуара.

### Warm human / Masha theme

- Опора.
- Честность.
- Тепло.
- Бытовая человеческая реальность.
- Может звучать как более мягкая версия летней темы.

### Wrongness stinger / short cues

Короткие сигналы для:

- отражения;
- сбоя звука;
- тихого шёпота;
- момента внутреннего провала;
- пустоты в кадре;
- странной паузы в обычном летнем шуме.

## Какие музыкальные задачи нужны первыми

- Main menu.
- Party.
- Ritual tension.
- Whisper motif.
- Morning / summer day.
- School tension.
- Investigation light.

## Что НЕ нужно в звуке

- Слишком хоррорные клише.
- Дешёвый эротический саунд.
- Перегруз драмой там, где нужна бытовая тишина.
- Треки, которые ломают ощущение летнего японского пространства.
- Слишком западный клубный вайб для подростковой вечеринки.

## Общие правила по звуку

- Не перегружать сцены музыкой.
- Мистика работает лучше через контраст с обычностью.
- Шёпоты и странности должны быть дозированными.
- У Маши и “человеческой линии” желательно свой более тёплый тон.
- Lust / romance / purity позже могут получить свои вариации одной общей мотивной базы.
