# Locations

## Зачем нужен этот файл

Документ помогает:

- понять, какие фоны реально нужны;
- не генерить лишнее;
- планировать day / evening / night-варианты;
- связывать локации со сценами и эмоциональными состояниями.

На текущем этапе действие предварительно происходит в японском городе или пригороде. Локации должны ощущаться реальными
для подростковой японской среды.

## Основные локации первого акта

### Комната героя

- Функция: база героя, место утреннего отката и одиночных мыслей.
- Контекст: комната японского старшеклассника в квартире или небольшом доме.
- Настроение: бытовая безопасность, которая больше не полностью безопасна.
- Время суток: утро, день, вечер.
- В кадре: кровать / футон или низкая кровать, стол, телефон, шкаф / зеркало, окно, школьная сумка.
- Важно для сцен:
    - утро после ритуала;
    - проверка отражения;
    - одиночество;
    - внутренний сдвиг.

### Квартира / место вечеринки

- Атмосфера: душная летняя подростковая вечеринка в японской квартире / арендованном помещении / доме знакомых.
- Кухня: напитки, упаковки из конбини, тесные разговоры.
- Коридор: переходы, ощущение тесноты, следы Леры.
- Общая комната: музыка, толпа, фоновая энергия.
- Отдельные фоны: кухня, коридор, общая комната.
- Вариациями можно закрыть: разные углы квартиры и шумные переходы.

### Ванная с ритуалом

- Ключевая локация первого мистического контакта.
- Контекст: маленькая японская ванная / умывальная зона с зеркалом.
- Важные элементы: зеркало, теснота, приглушённый свет, свеча, закрытая дверь.
- Настроение: бытовое место, которое становится неправильным.
- Нужна для раннего мистического CG:
    - отражение не совпадает;
    - свеча;
    - первое ощущение присутствия.

### Школьный коридор / школа

- Функция: социальное давление и попытка жить обычно.
- Контекст: японская старшая школа.
- Настроение: шум, взгляды, неловкость, сравнение себя с другими.
- Варианты:
    - коридор;
    - класс;
    - лестница;
    - окно;
    - дверь класса;
    - обувные шкафчики.
- Важно: японская школьная среда должна давить социально, а не выглядеть нейтральным фоном.

### Остановка / транспорт

- Функция: бытовая тревога вне ритуала.
- Варианты: автобусная остановка, поезд / электричка, станция, платформа.
- Нужно для: `first_whispers`, `lera_followup`.
- Время: день и вечер.
- Важные элементы: стекло, отражения, толпа, телефон, объявления, городской шум.

### Сквер / лавка / фонтан

- Функция: разговорные сцены и первые расследовательские обсуждения.
- Контекст: небольшой городской парк / pocket park / тихий сквер рядом с жилым районом.
- Настроение: летняя усталость, бытовая открытость, спокойное напряжение.
- Использование: разговор с Машей, сбор фактов, личные сцены.
- Желательно иметь: день и вечер.

### Спортивная площадка / двор

- Функция: приватный разговор с Артёмом.
- Контекст: маленькая спортивная зона у жилого района или школьная площадка после занятий.
- Настроение: подростковая бытовая реальность на фоне странности.
- В кадре: лавка, сетка, турники / баскетбольное кольцо, вечерний свет.
- Важно: место должно быть простым и узнаваемым.

### Набережная / река

- Функция: летняя свобода, прогулки, размышления, возможные романтические сцены.
- Контекст: река / канал / городская набережная в японском пригороде.
- Настроение: пространство и выбор.
- Потенциал: romance-сцены, внутренние монологи, летние развилки.
- Варианты: день, вечер, мягкий закат.

### Станция / платформа

- Функция: переходы, ожидание, одиночество среди людей.
- Настроение: обычная японская повседневность с возможностью тихого сбоя.
- Варианты: день, вечер, почти пустая платформа.
- Важные элементы: табло, линии на полу, объявления, стекло, отражения.

### Конбини

- Функция: бытовые встречи, короткие разговоры, летние покупки.
- Настроение: яркий свет, холодный воздух, обычность, которая может стать тревожной.
- Варианты: день, ночь, вечер после жары.
- Важные элементы: напитки, мороженое, журналы, касса, стеклянные двери.

### Вечерняя улица / городской переход

- Функция: сцены после встреч, одиночные мысли, возможные следы Леры.
- Настроение: влажный вечерний воздух, мягкий свет, редкие прохожие.
- Важные элементы: светофор, рекламные вывески, отражения в стекле.

### Жилой район вечером

- Функция: возвращение домой, разговоры, ощущение “обычной жизни”.
- Настроение: окна квартир, цикады / летний шум, тёплый воздух, тревожная тишина.
- Варианты: мягкий вечер, поздний вечер, пустая улица.

## Какие локации нужны в первую очередь

Приоритеты для first playable slice:

1. Комната героя в японском доме / квартире.
2. Место вечеринки в японской квартире / доме.
3. Ванная с зеркалом.
4. Японская школа: коридор / класс.
5. Остановка / транспорт / станция.
6. Сквер / маленький городской парк.
7. Двор / спортплощадка жилого района.

## Какие вариации желательно иметь

- День.
- Вечер.
- Ночь.
- Пустая / с людьми.
- Сухая жара / влажный вечер.
- Обычная версия / слегка тревожная версия.
- Реалистичная версия / cinematic anime atmosphere-версия с усиленным светом.
