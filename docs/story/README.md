# Story Documentation

`docs/story` — входная папка сюжетной документации `How I Fall`.

На текущем этапе здесь ведётся основная pre-production работа: структура истории, сцены, персонажи, переменные,
визуальное и production-планирование.

## Структура

- `core/` — базовые системные документы истории:
    - мир;
    - персонажи;
    - переменные и логика маршрутов.

- `planning/` — production- и art-планирование:
    - визуальный стиль;
    - локации;
    - ассеты;
    - звук;
    - стратегия генерации арта и промптов.

- `scenes/` — сцены, организованные по игровым дням.

## Рекомендуемый порядок входа

1. `README.md`
2. `workflow.md`
3. `flow.md`
4. `core/world.md`
5. `core/characters.md`
6. `core/variables.md`

## Сцены по дням

- `scenes/day_00_prologue/` — пролог, вечеринка, первый контакт и ранние route-ответвления.
- `scenes/day_01_afterparty/` — утро после, первые шёпоты, школа, летние дни, начало расследования.
- `scenes/day_02_tbd/` — будущий второй игровой день.
- `scenes/day_03_tbd/` — будущий третий день и далее.
