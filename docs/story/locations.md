# Locations

## Зачем нужен этот файл

Документ помогает:

- понять, какие фоны реально нужны;
- не генерить лишнее;
- планировать day / evening / night-варианты;
- связывать локации со сценами и эмоциональными состояниями.

## Основные локации первого акта

### Комната героя

- Функция: база героя, место утреннего отката и одиночных мыслей.
- Настроение: бытовая безопасность, которая больше не полностью безопасна.
- Время суток: утро, день, вечер.
- В кадре: кровать, стол, телефон, шкаф / зеркало, окно.
- Важно для сцен:
    - утро после ритуала;
    - проверка отражения;
    - одиночество;
    - внутренний сдвиг.

### Квартира / место вечеринки

- Атмосфера: душная летняя вечеринка, шум, алкоголь, подростковый хаос.
- Кухня: разговоры, напитки, первые социальные сцены.
- Коридор: переходы, ощущение тесноты, следы Леры.
- Общая комната: музыка, толпа, фоновая энергия.
- Отдельные фоны: кухня, коридор, общая комната.
- Вариациями можно закрыть: разные углы квартиры и шумные переходы.

### Ванная с ритуалом

- Ключевая локация первого мистического контакта.
- Важные элементы: зеркало, теснота, приглушённый свет, свеча, закрытая дверь.
- Настроение: бытовое место, которое становится неправильным.
- Нужна для раннего мистического CG:
    - отражение не совпадает;
    - свеча;
    - первое ощущение присутствия.

### Школьный коридор / школа

- Функция: социальное давление и попытка жить обычно.
- Настроение: шум, взгляды, неловкость, сравнение себя с другими.
- Варианты:
    - коридор;
    - класс;
    - лестница / окно / дверь класса.
- Важно: школа должна давить социально, а не выглядеть нейтральным фоном.

### Остановка / маршрутка

- Функция: бытовая тревога вне ритуала.
- Нужно для: `first_whispers`, `lera_followup`.
- Варианты: дневная маршрутка, вечерняя остановка.
- Важные элементы: стекло, отражения, толпа, телефон, городской шум.

### Сквер / лавка / фонтан

- Функция: разговорные сцены и первые расследовательские обсуждения.
- Настроение: летняя усталость, бытовая открытость, спокойное напряжение.
- Использование: разговор с Машей, сбор фактов, личные сцены.
- Желательно иметь: день и вечер.

### Спортивная площадка / двор

- Функция: приватный разговор с Артёмом.
- Настроение: обычная подростковая реальность на фоне странности.
- В кадре: лавка, турники, сетка, двор, вечерний свет.
- Важно: место должно быть простым и узнаваемым.

### Набережная / река

- Функция: летняя свобода, прогулки, размышления, возможные романтические сцены.
- Настроение: пространство и выбор.
- Потенциал: romance-сцены, внутренние монологи, летние развилки.
- Варианты: день, вечер, мягкий закат.

## Какие локации нужны в первую очередь

Приоритеты для first playable slice:

1. Комната героя.
2. Место вечеринки.
3. Ванная.
4. Школьный коридор.
5. Остановка / маршрутка.
6. Сквер / фонтан.
7. Двор / площадка.

## Какие вариации желательно иметь

- День.
- Вечер.
- Ночь.
- Пустая / с людьми.
- Сухая жара / мягкий вечер.
- Обычная версия / слегка тревожная версия.
